CS50 Web Programming

CS50's Web Programming with Python and JavaScript

Course Description

CS50W dives into the design and implementation of web apps with Python, JavaScript, and SQL using frameworks like Django, React, and Bootstrap. Topics include database design, scalability, security, and user experience. Through lectures and projects you learn to write and use APIs, create interactive UIs, and leverage cloud services like GitHub. By the end of the course, you gain knowledge and experience in principles, languages, and tools that make designing and deploying applications on the Internet possible.
