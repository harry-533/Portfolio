Cisco Networking Academy

Cisco's Computer Network Fundamentals

Course Description

The Cisco Networking Academy's Networking Essentials course provides learners with a broad foundational understanding of networking. This course taught the skills needed to understand the basics of a small and large scale network as well as how to set up the hardware and software needed to create a wired and wireless network.
