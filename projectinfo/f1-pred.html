Formula 1 Predictor

Predict the outcome of F1 races using historic data of past races

Project Description

The Formula 1 predictor website I created makes use of the Ergast API to gather current and historic data to make a prediction of each drivers qualifying and race position for upcoming races in the Formula 1 season. This project, made using django, allows the user to select a track and then using data such as current season average position, past results at the specific track and the results of the constructor, an algorithm I made takes the data, with some data being weighted more heavily to make a prediction. The Ergast API will be deprecated at the end of 2024 so the functionality of the website may not work past then.
