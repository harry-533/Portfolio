Sports Schedule

Weekly schedule with the aim of helping you never miss your favourite sports

Project Description

The Sport Schedule website is the perfect way to never miss your favourite sports by displaying the weeks upcoming fixtures for three sports currently, F1, Football and NFL. Using the API-SPORTS API the events are displayed on a calendar giving you all the information you need like what sport it is the details like teams and location, the start times and the duration. The website makes use of JavaScript to up date the calendar so it is always accurate and allow for events happening at the same time to be compiled into so they can be displayed in a neat list.
