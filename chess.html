Chess Engine

A Chess Engine that allows for player vs player, aswell as against an ai bot

Project Description

After completing CS50's introduction to AI course I decided that a good project would be to create a chess engine using python that is functional for player vs player on pygame, but with the added complexity of creating an AI bot that you can play against. the AI uses Negamax algorithm with alpha-beta pruning and the depth of recursion can be increased for a better bot (currently set at 2 due to computing power).
